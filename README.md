# Well Log QC Dashboard

An automated quality control dashboard for LAS well log files built with Python and Streamlit.

## What it does
- Parses LAS 2.0 well log files
- Automatically detects missing data, depth gaps, spikes, and flat-line segments
- Visualises flagged intervals on interactive log tracks
- Exports a QC report as CSV or PDF
